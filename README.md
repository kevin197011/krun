# sh
sh

## install
```bash
curl -o /usr/bin/krun https://raw.githubusercontent.com/kevin197011/sh/main/krun && chmod +x /usr/bin/krun
```

## usage
```bash
[root@localhost ~]# krun list
- hello-world.sh
- install-cfssl.pl
- install-ruby.sh
- set-vim-paste-mode.sh
[root@localhost ~]# krun hello-world.sh
hello world
```

## update
```bash
krun update
```
