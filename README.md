## install
```bash
curl -o /usr/bin/krun https://raw.githubusercontent.com/kevin197011/sh/main/krun && chmod +x /usr/bin/krun
```

## usage
```bash
[root@localhost ~]# krun list
shell name list:
- hello-world
- install-ruby
- set-vim-paste-mode
[root@localhost ~]# krun hello-world
hello world
```

## update
```bash
krun update
```
